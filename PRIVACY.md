# Privacy Policy

## Overview

This project does not collect or store personal data beyond what is required for its core functionality.

## Data Collection

The bot stores limited data necessary for operation, including:

* Guild (server) IDs
* User IDs
* TTS-related settings (e.g. nickname, voice, pitch, speed, selected language/region)

## Message Content

This project does **not** store, log, or analyze message content.

## Data Usage

Stored data is used solely to provide per-user and per-server functionality (such as TTS configuration).

## Data Sharing

* Self-hosted instances: All data remains within your own environment.
* Officially hosted bot: Data is not sold, shared, or distributed to third parties.

## Contact

For questions or concerns, open an issue on this repository.
